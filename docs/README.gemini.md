# Gemini CLI Usage

Install into the shared skill directory:

```bash
npx skills add gigio1023/godot-best-practice --agent gemini
```

Manual install:

```bash
mkdir -p ~/.agents/skills
git clone git@github.com:gigio1023/godot-best-practice.git ~/.agents/skills/godot-best-practice
```

Use this skill as local context for Godot 4.x best practices, official-doc lookup, implementation, version upgrades, and validation.
