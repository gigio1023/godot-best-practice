# Claude Code Usage

Install:

```bash
npx skills add gigio1023/godot-best-practice --agent claude-code
```

Manual install:

```bash
mkdir -p ~/.claude/skills
git clone git@github.com:gigio1023/godot-best-practice.git ~/.claude/skills/godot-best-practice
```

Use this skill for Godot 4.x GDScript, C#, scenes, resources, 2D, 3D, UI, physics, navigation, rendering, shaders, import/export, version upgrade, and runtime evidence work.
