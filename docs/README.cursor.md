# Cursor Usage

Install:

```bash
npx skills add gigio1023/godot-best-practice --agent cursor
```

Manual install:

```bash
mkdir -p ~/.cursor/skills
git clone git@github.com:gigio1023/godot-best-practice.git ~/.cursor/skills/godot-best-practice
```

Use this skill when Cursor is editing Godot 4.x projects and should follow Godot-native best practices with official-doc-backed details.
